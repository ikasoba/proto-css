Proto.CSS

About

Proto.CSS is a CSS framework that provides simplified styling for the lazy.
A simple and small JSX compiler is also included, so feel g free to use it.

How to use Proto.CSS

Proto.CSS is located at ./src/proto.css.
In the following example, we will actually use Proto.CSS after downloading Proto.CSS using git. 


  $ git clone https://github.com/ikasoba/proto-css.git
  $ touch index.html
        

  <!-- index.html -->
  <html>
    <head>
      <link rel="stylesheet" href="./proto-css/src/proto.css" />
    </head>
    <body>
      <button>
        count: 0
      </button>
      <script>
        let i = 0
        const button = document.querySelector("button")
        button.addEventListener("click", () => {
          i++
          button.innerText = `count: ${i}`
        })
      </script>
    </body>
  </html>

How to use the JSX compiler

The JSX compiler and loader are ./src/js/ directory.
The following example imports the loader and actually compiles the JSX on the client side. 


  <html>
    <head>
      <link rel="stylesheet" href="./proto-css/src/proto.css" />
      <script defer src="./src/js/jsx-loader.js" />
    </head>
    <body>
      <div id="root"></div>
      <!-- The type attribute can also be "text/babel". -->
      <script type="jsx" factory="h">
        import { h, Component, render } from 'https://esm.sh/preact'
        import { useState } from "https://esm.sh/preact/hooks"
          
        function Counter(props) {
          const [count, setCount] = useState(0)
          return (
            <button onClick={() => setCount(count + 1)}>
              count: {count}
            </button>
          );
        }

        render(<Counter />, document.getElementById("root"));
      </script>
    </body>
  </html>
